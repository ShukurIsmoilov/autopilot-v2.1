👋 Hello there,😻 this "Autopilot" program, but a bit modified (with support of arguments). It's easy to code on 📝, but we have to practice in 🖥️. Give it a try and have fun 🎉, after that by using provided commands get the car from 🎌 to 🏁. Note that, now we have obstacles on the way. Do not forget semicolon(;) in the end of commands. It has only 4 commands. Input field has example of code, try to run it. List of commands are listed below ⬇️

  • car.forward(arg);
  • car.backwards(arg);
  • car.right(arg);
  • car.left(arg);
